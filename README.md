# gin-lime
This is gin(Go) sample project.

## API
### Error
```json
{
  "status": 400,
  "message": "Bad Request"
}
```

### Todo
- `GET /api/v1/todo`
  - get all todo
- `GET /api/v1/todo/:id`
  - get todo by id
- `POST /api/v1/todo`
  - create todo
- `PUT /api/v1/todo/:id`
  - update todo by id
- `DELETE /api/v1/todo/:id`
  - delete todo by id

### User
- `GET /api/v1/user`
  - get all user
- `GET /api/v1/user/:id`
  - get user by id
- `POST /api/v1/user`
  - create user
- `PUT /api/v1/user/:id`
  - update user by id
- `DELETE /api/v1/user/:id`
  - delete user by id
